# Crop Recommendation System

A machine learning-based application that recommends suitable crops based on soil and environmental conditions.

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, Streamlit, Matplotlib, Seaborn

## Models Used
Decision Tree, Random Forest, Support Vector Machine (SVM), Logistic Regression, Naive Bayes

## Workflow
- Data preprocessing and cleaning  
- Feature selection  
- Model training and evaluation  
- Deployment using Streamlit  

## How to Run
1. Clone the repository:
   git clone https://github.com/sonaliahirwar16/crop-recommendation-system.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   streamlit run app.py
